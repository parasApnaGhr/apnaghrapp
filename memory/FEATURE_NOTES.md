## For Sellers (Calling Agents)

### 1. Daily Start & End Reports (Mandatory)

**Daily Start Report** - Required after every login:
- Upload office selfie (optional)
- Enter today's plan
- Set planned visits count
- Set expected deals count

**Daily End Report** - Required before logout:
- Enter clients called
- Enter visits booked
- Enter deals closed
- Enter properties shared
- Plan tomorrow's visits

**Warning Rules:**
- If Calls < 60 → Warning flag
- If Shares < 20 → Warning flag
- Warnings appear on your next login

---

### 2. Client Verification System (24-Hour Rule) - NEW

**What is it?**
After you share a property with a client, you must update the client's status within 24 hours.

**How it works:**
1. Share property with client via WhatsApp/SMS
2. After 24 hours, a verification popup will appear
3. You MUST verify each client before sharing more properties

**Verification Options:**
| Status | When to use |
|--------|-------------|
| **Closed Won** | Client converted, deal successful |
| **Closed Lost** | Client not interested |
| **Still in Progress** | Still following up (add notes) |

**IMPORTANT - Share Lock:**
- If you have pending verifications older than 24 hours → Share button is LOCKED
- Complete all verifications to unlock sharing

**IMPORTANT - Account Lock:**
- If you mark >10 clients as "Closed Lost" in 7 days → Account is LOCKED
- Only Admin can unlock your account
- Contact Admin immediately if locked

### 3. Performance Scoring System

**How Points are Calculated:**
| Activity | Points |
|----------|--------|
| Property Shared | +1 point |
| Visit Booked | +5 points |
| Deal Closed | +20 points |

**Bonus Points:**
- 20+ shares in a day: +10 points
- 5+ visits in a day: +15 points
- 1+ deal in a day: +25 points

**Penalty Points:**
- < 20 shares: -10 points
- No login: -50 points
- Late login (after 11 AM): -10 points

**Performance Tags:**
| Monthly Score | Tag |
|---------------|-----|
| 500+ | Top Performer |
| 300-500 | Good |
| 150-300 | Average |
| < 150 | Low (At Risk) |

### 2. Seller Performance Management

**Location:** Admin Dashboard → Seller Performance tab

**Available Tabs:**

#### a) Daily Tracking
- View all sellers' daily activity
- See login/logout times
- Track calls, shares, visits, deals
- View daily scores
- Identify warnings and late logins

#### b) Leaderboard
- Daily top performers
- Monthly top performers
- Performance tags visible

#### c) Earnings & Payout
- View all seller earnings
- Performance bonus calculation
- High performer bonus
- Mark sellers as "Paid"
- Bulk mark all pending as paid

#### d) Locked Sellers - NEW
- View all locked seller accounts
- See lock reason and date
- See "Closed Lost" count
- **Unlock Account** button to reactivate sellers

#### e) Motivation Quotes
- Add daily motivation quotes
- Set specific date for quotes
- Delete old quotes

---

### 3. Unlock Locked Seller Accounts - NEW

**When a seller is locked:**
1. Go to Admin Dashboard
2. Click "Seller Performance" tab
3. Click "Locked Sellers" tab
4. Find the locked seller
5. Click "Unlock Account" button

**Locking Rule:**
- Automatic lock when seller marks >10 clients as "Closed Lost" in 7 days
- Only Admin can unlock
